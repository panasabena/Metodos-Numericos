## Métodos Numéricos - Capítulo 1

### Errores Numéricos

* Errores de Redondeo:
Capacidad limitada de almacenamiento de las computadoras.

* De truncamiento:
Diferencia entre la formulación matemática exacta y la aproximación numérica.
